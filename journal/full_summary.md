# 전체 개발 히스토리 요약

## 주요 작업 목록
1. 터미널에서 다양한 명령어를 실험하며 터미널의 기능을 익혔습니다.
2. 디렉토리 생성 및 삭제, 파일 생성 및 삭제, 내용 출력, 폴더 이동 등의 작업을 수행했습니다.
3. `find` 명령어를 사용하여 파일 및 디렉토리를 검색하고, `locate` 명령어를 사용하여 파일을 검색했습니다.
4. 텍스트 편집기인 `vim`을 설치하고 사용했습니다.
5. 환경 변수를 설정하고, 별칭을 생성하고 삭제했습니다.

## 사용한 기술 스택
- bash
- Linux 명령어 (ls, cd, mkdir, rm, cp, etc.)
- vim
- locate
- find
