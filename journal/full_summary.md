# 전체 개발 히스토리 요약

## 📋 주요 작업 목록
* 터미널에서 다양한 명령어를 실행하여 파일과 디렉토리를 생성 및 관리했습니다.
* `touch`, `su`, `echo`, `exit`, `clear`, `ls` 등의 명령어를 사용했습니다.
* `cd` 명령어를 사용하여 디렉토리를 이동했습니다.
* `mkdir` 명령어를 사용하여 디렉토리를 생성했습니다.
* `rm` 명령어를 사용하여 파일과 디렉토리를 삭제했습니다.
* `cp` 명령어를 사용하여 파일을 복사했습니다.
* `vim` 및 `nano` 편집기를 사용하여 파일을 수정했습니다.
* Git을 사용하여 커밋을创建하고, 로그를 확인했습니다.
* Python, JavaScript/React, CSS, HTML 파일을 수정했습니다.
