# ai-summarizer 프로젝트 기록

## 📌 프로젝트 개요
ai-summarizer 프로젝트는 Groq API 기반의 텍스트 요약기를 개발하는 프로젝트입니다. 이 프로젝트는 사용자에게 문서의 요약본을 제공하는 것을 목표로 합니다.

## 📋 작업 히스토리
작업 히스토리는 다음과 같습니다. 
- viewer.css 경로 수정
- summarize_file 추가 및 FileNotFoundError 처리
- 청킹/KeyError 안전처리
- .env API 키 추가 및 gitignore 처리
- requirements.txt 추가

## 🛠 사용한 기술 스택
Python: 2개 파일
Shell: 1개 파일
CSS: 2개 파일
HTML: 1개 파일

## 🐛 트러블슈팅
트러블슈팅은 다음과 같습니다.
- FileNotFoundError 처리
- 청킹/KeyError 안전처리
- 401/KeyError 수정

## ⚠️ 커밋 메시지 개선 제안
커밋 메시지 "수정"은 너무 짧습니다. 더 자세한 설명을 추가할 필요가 있습니다.
