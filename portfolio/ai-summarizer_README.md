# ai-summarizer 프로젝트

## 📌 프로젝트 소개
ai-summarizer 프로젝트는 Groq API 기반의 텍스트 요약기를 개발하는 프로젝트입니다. 이 프로젝트는 사용자에게 문서의 요약본을 제공하는 것을 목표로 합니다.

## 🎯 개발 동기
개발 동기는 사용자에게 문서의 요약본을 제공하여 시간과 노력을 절약해주는 것입니다. 또한, 문서의 주요 내용을 빠르게 파악할 수 있도록 도와주는 것입니다.

## 🛠 기술 스택
사용한 기술 스택은 다음과 같습니다.
- Python: 2개 파일
- Shell: 1개 파일
- CSS: 2개 파일
- HTML: 1개 파일

## ✨ 주요 기능
주요 기능은 다음과 같습니다.
- 문서의 요약본을 제공하는 기능
- 파일을 읽어서 요약 생성을 처리하는 기능
- 청킹/KeyError 안전처리 기능
- 401/KeyError 수정 기능

## 🔥 트러블슈팅
트러블슈팅은 다음과 같습니다.
- FileNotFoundError 처리
- 청킹/KeyError 안전처리
- 401/KeyError 수정

## 📈 성능 개선
성능 개선은 다음과 같습니다.
- 코드를 리팩토링하여 더 효율적이고 안정적인 코드를 만들었습니다.
- 에러를 해결하고, 코드를 수정하는 방법을 배웠습니다.

## 💡 배운 것들
배운 것들은 다음과 같습니다.
- 파일을 읽어서 요약 생성을 처리하는 방법을 배웠습니다.
- FileNotFoundError와 ValueError를 처리하는 방법을 배웠습니다.
- Python의 문자열.replace() 함수를 사용하여 문자열을 변경하는 방법을 배웠습니다.
