# ai-summarizer

## 📌 프로젝트 소개
ai-summarizer 프로젝트는 문서를 요약하여 주요 내용을 간단하게 제공하는 것을 목표로 합니다. 이 프로젝트는 문서 요약 기능을 제공하여 사용자가 문서의 주요 내용을 빠르게 이해할 수 있도록 도와줍니다.

## 🛠 기술 스택
이 프로젝트는 주로 Python을 사용하여 개발되었습니다. Python의 requests 라이브러리를 사용하여 HTTP 요청을 보내고, 에러 처리를 하였습니다. 또한, Git을 사용하여 버전 관리를 하였습니다.

## ✨ 주요 기능
ai-summarizer 프로젝트의 주요 기능은 문서를 요약하여 주요 내용을 제공하는 것입니다. 문서를 읽어서 요약 생성을 처리하는 함수를 추가하여 파일을 읽어서 요약 생성을 처리할 수 있습니다. 또한, `FileNotFoundError`와 `ValueError`를 처리하여 안정적인 동작을 보장합니다.

## 🔥 개발 과정에서 해결한 문제들
개발 과정에서 발생한 문제들 중 하나는 API 오류 처리와 requests.get()에서 발생하는 예외 처리였습니다. 이러한 문제들을 해결하여 안정적인 동작을 보장하였습니다. 또한, `FileNotFoundError`와 `ValueError`를 처리하여 문서를 읽어서 요약 생성을 처리할 때 발생할 수 있는 오류를 해결하였습니다.
