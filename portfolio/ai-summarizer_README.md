# ai-summarizer

## 한 줄 소개
ai-summarizer 프로젝트는 Groq API 기반의 텍스트 요약기를 개발하는 프로젝트입니다. 이 프로젝트는 텍스트를 요약하여 중요한 정보를 빠르게 파악할 수 있도록 도와줍니다.

## 기술 스택
ai-summarizer 프로젝트는 다음과 같은 기술 스택을 사용합니다.
- Python: 4개 파일
- Shell: 1개 파일
- CSS: 2개 파일
- HTML: 1개 파일

## 주요 기능
ai-summarizer 프로젝트의 주요 기능은 다음과 같습니다.
- 텍스트 요약: 텍스트를 요약하여 중요한 정보를 빠르게 파악할 수 있도록 도와줍니다.
- 파일 요약: 파일을 읽어서 요약 생성을 처리합니다.
- 에러 처리: FileNotFoundError, ValueError, KeyError 등 다양한 에러를 처리합니다.

## 트러블슈팅
ai-summarizer 프로젝트에서 발생한 트러블슈팅은 다음과 같습니다.
- FileNotFoundError 처리
- ValueError 처리
- KeyError 안전처리
- 401/KeyError 수정
- viewer.css 경로 수정
